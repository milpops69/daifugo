# DaiFugo 🃏

Пиксельная карточная игра **Daifugo** (大富豪) для 4 игроков (вы + 3 бота).

---

## Структура проекта

```
daifugo-project/
├── index.html   — HTML-разметка (экраны: заголовок, правила, игра)
├── style.css    — Вся стилизация (розово-чёрная пиксельная тема)
├── cards.js     — Рендерер карт на Canvas (пиксель-арт)
├── game.js      — Вся игровая логика (ИИ, drag&drop, анимации)
├── server.py    — Простой локальный сервер для запуска
└── README.md    — Этот файл
```

---

## Быстрый запуск

### Вариант 1 — Python сервер (рекомендуется)

```bash
python server.py
```

Игра автоматически откроется в браузере на `http://localhost:8000`.

### Вариант 2 — VS Code Live Server

1. Установите расширение **Live Server** (ritwickdey.LiveServer)
2. Откройте `index.html`
3. Нажмите **Go Live** в правом нижнем углу

### Вариант 3 — Открыть напрямую

Просто откройте `index.html` в браузере (Chrome/Firefox/Edge).  
> ⚠️ Некоторые браузеры блокируют локальные файлы — используйте сервер.

## Правила игры

| Карта | Сила |
|-------|------|
| 3     | слабейшая |
| 4–K   | по возрастанию |
| A     | сильная |
| 2     | очень сильная |
| Joker | сильнейшая |

- **Ход**: кладите карты той же силы или выше. Количество карт должно совпадать (пара → пара).
- **Пас**: нет хода — пасуйте. Все спасовали — стол очищается.
- **8-стоп**: восьмёрка сбрасывает стол, тот же игрок ходит снова.
- **Революция**: 4 карты одного ранга — порядок сил переворачивается.
- **Джокер**: подставляется вместо любой карты.

### Очки

| Место     | Ранг    | Очки |
|-----------|---------|------|
| 1-е место | Богач   | +3   |
| 2-е место | Богатый | +2   |
| 3-е место | Бедняк  | +1   |
| 4-е место | Нищий   | +0   |

## Управление

- **Перетащить карту** мышью на стол — сыграть
- **Клик** по карте — выбрать/снять выбор
- Выбрать несколько карт кликом, затем перетащить одну — сбросить все выбранные

---

## Разработка

Проект состоит из чистого HTML/CSS/JS — никаких фреймворков и зависимостей.

- `cards.js` — изолированный модуль рендера карт, не зависит от игровой логики
- `game.js` — вся логика, использует функции из `cards.js` (`makeCard`, `makeBack`)
- Для редактирования стилей — только `style.css`
